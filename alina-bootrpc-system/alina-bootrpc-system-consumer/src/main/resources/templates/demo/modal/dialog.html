Setting Options

Relevant functions of modal windows can be implemented through data binding or Javascript. If you use data binding, you can add elementsdata-，eg:data-backdrop=""。

Name Type Default Description
backdrop boolean or string 'static' true Mask layer, or use'static'Specified mask layer is not associated with closed modal window
keyboard boolean true Exit modal window when pressing Esc
show boolean true Show modal window after initialization
remote path false

Data binding is recommended, or use jQuery.load

Remote URL example：

<a data-toggle="modal" href="remote.html" data-target="#modal">Click me</a>
